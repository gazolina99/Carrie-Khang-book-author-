Production env vars (Railway, Render, VPS, etc.)

Required:

  DATABASE_URL=file:./dev.db
    (single server with persistent disk — OR a Postgres URL if you use Postgres)

  AUTH_SECRET=long-random-string

  ADMIN_EMAIL=author@example.com
  ADMIN_PASSWORD=your-secure-password

  AUTH_URL=https://your-domain.com
  NEXT_PUBLIC_SITE_URL=https://your-domain.com

Optional — auto migrate + seed on server start:

  RUN_DB_SETUP_ON_BOOT=1

Build: npm run build
Start: npm run start

Sign in: https://your-domain.com/login

This repo runs on GitHub for source control.
Production = any Node 20 + PostgreSQL host (see README).

Quick local:
  1. npm install
  2. Copy .env.example to .env and fill DATABASE_URL + AUTH_SECRET
  3. npx prisma migrate deploy && npm run db:seed
  4. npm run dev → http://localhost:3000

Copy-paste env names: DEPLOY-ENV-VARS.txt
